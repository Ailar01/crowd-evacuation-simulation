# Crowd Evacuation Simulation

## Description

This project is a simple crowd evacuation simulation developed in C++ using the Raylib library.

The simulation represents multiple agents moving towards an exit while trying to avoid collisions with each other. It also compares two different collision detection methods and displays real-time performance statistics.

## Features

- Crowd evacuation simulation
- Collision avoidance between agents
- Naive collision detection
- Grid-based optimized collision detection
- Real-time FPS display
- Collision checks counter
- Grid cells usage statistics
- Agent speed control
- Pause and resume simulation
- Reset simulation
- Add or remove agents

## Controls

- `1` - Naive collision mode
- `2` - Grid optimized mode
- `UP` - Add 100 agents
- `DOWN` - Remove 100 agents
- `G` - Change agent speed
- `R` - Reset simulation
- `SPACE` - Pause / Resume

## Technologies Used

- C++
- Raylib
- Spatial Grid Optimization

## Purpose

The purpose of this project is to simulate crowd movement and compare different collision detection approaches while monitoring their performance in real time.
